### 2. 抓取内容在招标列表页显示乱码 ✅ 已修复

**现象:** 从吉林省公共资源交易中心爬取的数据，标题字段包含HTML标签（如`<font color='#0066FF'>`），在列表页显示为乱码。

**原因分析:**
- API返回的`title`字段包含HTML格式标签
- 前端直接显示原始内容，未进行HTML清理

**解决方案:**
- 在 `DynamicSpider._extract_api_fields()` 方法中添加了 `_clean_html_tags()` 辅助方法
- 使用 BeautifulSoup 清理 HTML 标签，失败时回退到正则表达式
- 对 `title` 和 `tenderer` 字段应用清理

**修复提交:** `dynamic.py` 第 514-575 行

**示例效果:**
```
原标题: <font color='#0066FF'>[电子化][竞争性磋商]</font>吉林省信息化建设促进中心-委托业务服务
清理后: [电子化][竞争性磋商]吉林省信息化建设促进中心-委托业务服务
```
